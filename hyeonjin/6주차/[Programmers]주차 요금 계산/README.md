## 로직

## 리뷰
- 구현 문제는 차근차근 하는게 중요한 것 같다.
- Math.ceil 함수 쓸때 나눗셈을 하면 자동으로 내림이 된다는 사실을 간과해서 헤맸다. 
- Math.ceil 을 쓸 때는 꼭(double)을 적용해 주어야 함을 기억해야겠다.
